# LLM-Transformer-Toolkit

From-scratch implementations of transformers, GPT models, and language model experiments. Built to deeply understand the architecture behind modern LLMs.

## What's Inside

- **tinygpt/** — Minimal GPT implementation for learning
- **basicllm/** — Basic language model experiments
- **transformer.py** — Core transformer architecture components
- **main.py** — Entry point for running experiments

## Tech Stack

- Python 3.10+
- PyTorch
- Tokenizers
- uv (package manager)

## Getting Started

```bash
git clone https://github.com/SaadRiaz99/LLM-Transformer-Toolkit.git
cd LLM-Transformer-Toolkit
pip install -e .
python main.py
```
